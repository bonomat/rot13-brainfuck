# ROT13 Cipher written in brainfuck

## Summary
This program enciphers its input with the [ROT13][3] cipher

## Usage
Assuming you have [bf][1] installed:

    cat plain.txt | bf rot13.bf
    and
    cat encrypted.txt | bf rot13.bf
